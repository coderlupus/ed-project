Este projeto compara árvores binárias, AVL e tabelas de hash.

Como compilar:
gcc arvore_binaria.c -o arvore_binaria
gcc arvore_avl.c -o arvore_avl
gcc tabela_hash.c -o tabela_hash

Como executar:
./arvore_binaria
./arvore_avl
./tabela_hash

Os tempos são salvos em arquivos .dat na pasta graficos.
Os gráficos são gerados com:
gnuplot script.gnuplot
